# Java Console Task Manager

A simple console-based Task Manager application built using core Java.  
This application allows users to manage daily tasks through a menu-driven interface and stores data persistently using file serialization.

---

## Features
- Add new tasks with priority and due date
- View all existing tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using file serialization

---

## Technologies Used
- Java
- Java Collections Framework
- File I/O and Serialization
- Object-Oriented Programming (OOP)

---
